# To-Do App (MERN Stack)

A full-stack **To-Do application** built with the **MERN stack**. This app allows users to **create, read, update, and delete tasks (CRUD operations)**. It features a clean and responsive UI with persistent storage using **MongoDB**.  

---

## Features

- Add new tasks  
- View all tasks  
- Update existing tasks  
- Delete tasks  
- Responsive and modern UI  
- Persistent storage using MongoDB  

---

## Technologies Used

- **Frontend:** React.js, Axios, Tailwind CSS / CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB   
- **Others:** CORS, dotenv for environment variables
